# Print Method for obwoe Objects

Displays a concise summary of optimal binning results, including the
number of successfully processed features and top predictors ranked by
Information Value.

## Usage

``` r
# S3 method for class 'obwoe'
print(x, ...)
```

## Arguments

- x:

  An object of class `"obwoe"`.

- ...:

  Additional arguments (currently ignored).

## Value

Invisibly returns `x`.
